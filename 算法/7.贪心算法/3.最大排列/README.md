# 最大排列

## 题目描述

给定一个1到N的随机排列数组A，最多可以进行B次交换，求能得到的字典序最大的排列 (越靠前的数字越大)

### 示例

- `[1, 3, 2], k = 1` 输出 `[3, 1, 2]`
- `[1, 3, 2], k = 2` 输出 `[3, 2, 1]`

## 模拟

### 例子1

假设给定 `[1, 2, 3, 4]`，`B: 1`

因为 4 最大，把它移到第一位，得到 `[4, 1, 2, 3]`

所以最终结果为 `[4, 1, 2, 3]`

### 例子2

假设给定 `A: [3, 2, 4, 1, 5]`, `B: 3`

1. 第一次交换，因为 5 最大，把它移到第一位，得到 `[5, 2, 4, 1, 3]`
2. 第二次交换，因为 4 最大，把它移到第二位，得到 `[5, 4, 2, 1, 3]`
3. 第三次交换，因为 3 最大，把它移到第三位，得到 `[5, 4, 3, 1, 2]`

所以最终结果为 `[5, 4, 3, 1, 2]`

## 规则总结

通过前面的例子可以发现，我们可以总结出以下规则：

- 我们的交换顺序为 `1 -> 2 -> 3 ->... -> N`，即从左到右，尽可能把最大的数字移到最前面
- 对于每一次交换，我们都可以把最大的数字移到最前面，这样就可以得到字典序最大的排列
- 我们只需要进行 `B` 次交换，就可以得到字典序最大的排列

## 代码实现

```c
static void quick_swap(int *a, int *b)
{
    /* 使用异或交换两个变量的值 */
    /* 原理: a ^ b ^ b = a */ 
    *a ^= *b;
    *b ^= *a;
    *a ^= *b;
}

void max_permutation(int *A, int n, int B)
{
    for (int i = 0; i < B; i++) // 进行B次交换
    {
        int max_index = i;
        for (int j = i + 1; j < n; j++) // 找出最大的数字的索引
        {
            if (A[max_index] < A[j])
                max_index = j;
        }
        quick_swap(A + i, A + max_index); // 交换最大的数字到最前面
    }
}
```
