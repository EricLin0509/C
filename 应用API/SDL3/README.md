# SDL3

**SDL3** (Simple DirectMedia Layer 3) 是一个跨平台的多媒体开发库，旨在为音频、键盘、鼠标、游戏手柄和图形硬件的低级访问提供统一的抽象接口。它是 SDL2 的继任者，专注于更现代的 C 语言特性、模块化设计以及提升开发者的使用体验

## 核心特性

-   **跨平台支持**：支持 Windows, macOS, Linux, iOS, Android 等主流平台
-   **硬件抽象**：提供对音频、2D/3D 图形（通过 OpenGL, Vulkan, Direct3D 等）、输入设备（键盘、鼠标、游戏手柄、触控）的统一访问
-   **现代化设计**：代码库采用更现代的 C 语言标准，API 设计更清晰、一致
-   **模块化**：核心功能被分解为更独立的模块，允许开发者按需链接，减少最终应用的体积
-   **增强的输入与事件系统**：改进的游戏手柄支持、传感器 API 以及更灵活的事件处理机制
-   **高性能**：专注于低开销，为游戏和高性能多媒体应用提供坚实基础
